# SQL
Simple SQL tables designed for clean, structured practice of core database commands and query fundamentals.

## 🛢️ SQL Practice Tables
Repository containing a set of basic SQL tables designed to practice fundamental commands: structure creation, data insertion, queries, filters, and relational operations.

## 📌 Description
This repository includes simple SQL scripts oriented toward learning exercises and quick testing. It features small, easy‑to‑understand tables ideal for practicing:

Table creation (CREATE TABLE)

Data insertion (INSERT INTO)

Basic queries (SELECT, WHERE, ORDER BY)

Simple relationships (JOIN)

Data modification (UPDATE, DELETE)

Aggregate functions testing (COUNT, SUM, AVG)

## 📁 Repository Structure
/tables/ → Table creation scripts

/data/ → Sample data to populate the tables

/queries/ → Practice queries and exercises

README.md → Project documentation

## 🎯 Objective
Provide a lightweight and clear environment to experiment with SQL, validate concepts, and strengthen modeling and querying skills.
